## Titanic 

**Description:**
Performed exploratory data analysis on the Titanic dataset using Python and pandas. Cleaned the data by handling missing values, dropping low-quality columns, and standardizing column names. Explored the data through filtering, string operations, and feature engineering. Aggregated survival rates by passenger class and gender using groupby and pivot tables, revealing that first class female passengers had the highest survival rate.
